# Creatorverse 2026 Website

This is the official website for Creatorverse 2026, built with Next.js and React.
It fully implements the provided design requirements and visual aesthetics.

## Features
- **Responsive Design**: Mobile, Tablet, and Desktop optimized.
- **Modern UI**: Dark theme with vibrant brand colors (Pink, Green, Purple).
- **Animations**: Entrance animations and floating shapes using Framer Motion.
- **Sections**:
  - Hero (Animated branding)
  - Why Creatorverse
  - Speakers (Coming Soon teasers)
  - Schedule (Interactive Folder/Tab look)
  - Tickets (Pricing Tiers)
  - FAQ & Blog
  - Footer (Newsletter & Links)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure
- `src/app/page.tsx`: Main landing page assembly.
- `src/app/layout.tsx`: Global layout, fonts (Inter/Outfit), and Navbar.
- `src/app/globals.css`: Global styles and CSS variables.
- `src/components/`: Reusable UI components.
